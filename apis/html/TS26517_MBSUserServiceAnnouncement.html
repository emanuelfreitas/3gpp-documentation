Discover User Service Descriptions

Discover User Service Descriptions that match the supplied query filter(s). At least one filter query parameter must be included in the request URL.

query Parameters
service-class
required
string (Uri)

Filter for User Service Descriptions tagged with the supplied service class term identifier expressed as a fully-qualified URI string from a controlled vocabulary

Responses

/user-service-descriptions

Retrieve User Service Description

path Parameters
externalServiceId
required
string

The external service identifier of a User Service provisioned in the MBSF.

Responses

/user-service-descriptions/{externalServiceId}
